# Мониторинг лесных пожаров при помощи ML-решений
### Актуальность и контекст
Согласно статистике (forbes.ru), площади лесных пожаров в России с начала года достигли 1,2 млн га — это в 2,5 раза выше показателей 2024 года. Несмотря на борьбу с лесными пожарами и профилактикой путем оповещения жителей страны в школах, институтах и так далее, количество пожаров продолжает расти.

# ML-система обнаружения пожаров
Данная система позволяет в "прямом эфире" обрабатывать изображения с камер, следящих за окружающей средой, и выдавать предупреждения о пожарах. Для более легкого понимания работы, в данном проекте предусмотрена загрузка фотографии с вашего устройства

# Инструкция по запуску
1. Подготовка рабочей среды
* Загрузите файл Project_code формата .ipynb (Jupiter notebook) на свое устройство
* Откройте данный файл на сайте google colab
* Установите зависимости, указанные в первой строчке кода

```!pip install -q inference-sdk opencv-python matplotlib Pillow requests```

2. Запустите код от начала до конца, в некоторых местах нужна будет дополнительная информация
  * Этап 2. Запустите блок, после чего вам будет необходимо выбрать изображение формата .jpg с вашего устройства
  * Этап 3. Для корректной работы вам нужно инициализировать работу клиента Roboflow и вставить ваш персональных API-ключ

3. После выполнения первых двух пунктов, вы сможете получить изображение, на котором будет выделена рамка с предполагаемым пожаром (если он есть), а также уверенность модели в правильности решения (вероятность)
